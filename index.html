HTML Syntax


1. Basic HTML Document structure:


<html> - Beginning of tht html-file

<head>
<meta name ="description" content="..."/>
<meta name ="keywords" content="..."/>
<meta name ="author" content="..."/>

- This lines tell Btowser what language we're writing in, what are tags, description of the site etc.

<title>.....</title> - Name of the site
<link rel="stylesheet" href="..."> - Link to the stylesheet
</head>

<body>

<header> - Header section
<nav> - Navigation menu
<ul>
<li><a href="...">Link1</a></li> - Links of the menu
<li><a href="...">Link2</a></li>
<li><a href="...">Link3</a></li>
</ul>
</nav>
</header>

<div class="Section1"> - Separate divs for each page-section
<h1>Title Goes Here</h1>
<img src="..." alt="..."></img>
</div>

<footer> - Footer section
<div class="social-icons">
<a href="..."></a>
<a href="..."></a>
<a href="..."></a>
</div>
</footer>

</body>

</html>

2. Html Tags



- Text tags:


  • <h1></h1> - Title (to h5) , <p></p> - Paragraph;

  • <abbr title="..."></abbr> - Longer description of a text when hover over;

  • <acronym title="..."></acronym> - Ecsplanation of a short text when hover over;

  • <b></b> - Bold text; <strong></strong> - Strong text;

  • <big></big> - Bigger text; <small></small> - Smaller text;

  • <blogquote></blogquote> - Important statement; <quote></quote> - Statement;

  • <i></i> , <em></em> - tags for Italic text;

  • <center></center> - Centered text; <s></s> - Line through text; </br> - Line break;

***********************************************************************************************************

- Image Tags:


  • <img></img> - Image;

  • src ="..." - Location of the files;

  • alt="..." - Alternative text;

  • height="..." - Height of the image;

  • width="..." - Width of the image;

***********************************************************************************************************

- List Tags:


  • <dl></dl> - Definition list;

    • In this list:

  • <dt></dt> - Title; <dd></dd> - Description;

  • <ol></ol> -Ordered list;

  • <ul></ul> - Unordered list;

  • <li></li> - List item in any list;

***********************************************************************************************************

- Link Tags:


  • <a></a> - Link indicator;

  • href="..." - Tells to open the link if clicked;

  • target="..." - Where new link will bw opened; In target:

  • 'blank' - open in current page; 'self'- open in a new page;

  • <a name="Some Name">Some Name</a> - allows to jump to certain text on page;

***********************************************************************************************************

- Table Tags:


  • <table></table> - Table indicator Tn the table:

  • summary="...." - Description;

  • <caption></caption> - Heading;

  • <tr></tr> - table row;

  • <th></th> - Header of a column;

  • <td></td> - Table data(in a column);

***********************************************************************************************************

- Form Tags:


  • <form action="mail.php" method="post"></form> - Indicate a Form and tells browser what to do with it;

  • method="post" - e-mail on which to send wisible, method="get" - invisible;

    • In the form tag:

  • <b>Heading Here</b></br> - Heading before name, email, password or other data section;

  • <input type="text" name="email" size=40/></br> - Email section

  • <input type="text" name="textinput"/></br> - Name(or other data) section;

  • <input type="password" name="password"/></br>

  • <textarea cols=40 rows=10 name="message">Message goes here</textarea></br> - Text Message section;

  • <input type="radio" name="radioinput" value="Love this"/>Love This</br> - Input to check among possible variations;

    • Love This
    Hate This

  • <input type="checkbox" name="checkboxinput" value="Hate it"/>Hate it</br> - Chechbox for multiple possibilities;

    • Hate it
    Love it

  • <select name="selectinput">
    <option value="loveselect">I Love This</option>\
    </select></br>

    • - This is a Dropdown checkbox;


  • <input type="submit" value="Send"> - Submit/Send button;

    • - Submit/Send button;

CSS Syntax

1. CSS-Selectors:


We take a part of HTML, using CSS-selectors and style it;

    - ID-Selectors - Styling a unique part of HTML.


  • Every id is used only 1 time.

  • In html-file we geve a sertain area unique ID this way:

    • <div id="testimonials">...</div>

  • In css-file we'll select it this way:

    • #testimonials {
    color: ...;
    };

    - Type-Selectors


  • Styling by HTML-Tags. Applies to all web pages

    • h1 {
    font-size: ...;
    color: ...;
    };

    - Class-Selectors


  • Reusable style blocks.

  • We can create a class and apply it to multiple elements on the page.

  • In html-file:

    • <h1 class="superHeading">My super heading</h1>

  • in css-file:

    • .superHeading {
    font-size: ...;
    color: ...;
    };
